# Smart Study Planner

Main architecture:

```text
React.js frontend
  ↓ REST API
Spring Boot backend
  ↓ JPA / Hibernate
MySQL database
```

Open the website at:

```text
http://localhost:5173
```

Spring Boot runs on `http://localhost:8090/api` so it does not conflict with Jenkins on `8080`.

For complete setup and startup commands, see [replit.md](./replit.md).
